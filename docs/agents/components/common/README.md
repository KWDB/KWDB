# Common Component Notes

The `common/` subtree holds shared C and C++ infrastructure that can affect many
modules at once.

Use these references first:

- `docs/agents/logging.md` for log behavior and compatibility constraints
- `docs/agents/errors.md` for error-code, message, and JSON compatibility
- `docs/agents/testing-flow.md` for widening verification when shared surfaces change

When this area gains repeated local patterns, add focused notes here instead of
growing the root `AGENTS.md`.
